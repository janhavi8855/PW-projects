# PHP Login System Project

## Setup Instructions

1. Install XAMPP
2. Start Apache and MySQL
3. Copy project folder into:
   C:\xampp\htdocs\
4. Import intern_db.sql in phpMyAdmin
5. Open browser:
   http://localhost/login_system/login.php

## Features

- User Registration
- Login & Logout
- Role-Based Access
- Admin Dashboard
- User Dashboard
- CRUD Operations
- Bootstrap Responsive UI
